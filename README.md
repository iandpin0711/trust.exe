## 👁️ Project Overview
**Trust.exe** is a 2D strategy and simulation game built in **Godot**. Unlike traditional technical hacking games, here the code is not the main character, but **Social Engineering**.

You must manipulate your victims through tense dialogues and fraudulent emails to extract credentials without raising suspicion.

### ✨ Key Features
* **🎭 Hybrid Infiltration:** A strategic mix of voice manipulation (calls) and digital attack deployment (phishing).
* **⚠️ Suspicion Management:** A dynamic risk/reward system. One mistake in your speech and the victim will hang up.
* **🎓 Cyber-Awareness:** Learn real scam tactics (typosquatting, urgency) while you play.
* **🌑 Corporate Noir Atmosphere:** A gritty and minimalist visual style that immerses you in the world of digital fraud.

## 🛠️ The Arsenal
To breach your target's security, you must master these two pillars:

| Category | Action / Asset | Special Role | Attribute Impact |
| :--- | :--- | :--- | :--- |
| **Dialogue** | 🎧 Tech Support Script | Initial trust building | 📉 Decreases suspicion significantly |
| **Dialogue** | 🚨 Urgent Threat | Pressure the victim | 📈 High urgency / Increases risk |
| **Phishing** | 📧 Typosquatted Email | Send malicious link | ✅ Effective only at high trust levels |
| **Terminal** | 💀 Hit List | Target selection | 🏆 Defines difficulty and rewards |
